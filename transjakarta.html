Stop Density Analysis

This heatmap shows stop concentrations. Brighter areas mean higher stop density and better transit access.

Route Speed Analysis

Displays average route speeds. Yellow = Slower, Red = Faster. This helps identify traffic congestion points.

Network Communities Analysis

Reveals the network's structure by identifying 752 distinct clusters (communities) of connected stops, each shown in a different color. It also highlights "bridge routes" that link these communities, like Route 3 (Kalideres - Monas).

Network Centrality Analysis

Identifies the most influential stops. Bridges (Blue), like Potlot, connect different network parts. Hubs (Orange), like Cibubur Junction, have the most direct connections.

Bivariate Transit Gap Analysis

This compares transit service with population density. Red = Low Service, Blue = High Service. Brightness shows population. Bright red areas are critical service gaps (high population, poor transit) that may need new routes.
