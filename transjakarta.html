Stop Density Analysis

Route Speed Analysis

Network Communities Analysis

Reveals the network's structure by identifying 752 distinct clusters (communities) of connected stops. It also highlights "bridge routes" that link these communities, like Route 3 (Kalideres - Monas).

Network Centrality Analysis

Identifies influential stops. Bridges (Blue), like Simpang Danau Sunter Selatan, connect network parts. Hubs (Orange), like Cibubur Junction, have the most direct connections.

Bivariate Transit Gap Analysis

This compares service with population. Red = Low Service, Blue = High Service. Brightness shows population. Bright red areas are critical service gaps (high population, poor transit) for prioritization.
